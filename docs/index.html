六种模式覆盖日常所有重命名场景

Aa

添加前缀 / 后缀

给所有文件名批量加上前缀或后缀文本。照片加日期、文件加版本号。

⇄

查找替换

把文件名里的指定文字批量换成别的。告别挨个手动改。

#

序号命名

自定义模板加自增编号。可以设置起始值、编号位数。

.ext

修改扩展名

一键把 .jpeg 全改成 .jpg，或者其他任意扩展名。

↶

一键撤销

改错了不慌。点一下撤销，所有文件恢复原名。

◉

实时预览

改名之前先看效果。绿色高亮显示哪些会变，确认了再执行。
